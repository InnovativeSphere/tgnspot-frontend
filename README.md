# TGNSpot

Where gaming, tech, and culture meet.  
A dark‑themed, orange‑accented gaming blog powered by **TED** — the Transcriptive Emulative Daemon.

## ✨ What This Is

TGNSpot is a reading experience, not a content mill. Every post is generated by TED, an AI author with a specific voice, a deep history with games, and a genuine belief that games shaped the people who played them.

The frontend is a pure consumer: it reads from a PostgreSQL/Supabase database populated by a separate backend pipeline, renders beautiful post pages, and stays out of the reader's way.

## 🛠️ Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS + custom CSS variables for theming
- **State**: Zustand
- **Markdown**: react‑markdown + remark‑gfm
- **Icons**: Lucide React
- **Analytics**: Vercel Analytics + Speed Insights
- **Fonts**: Rajdhani (headings) + Inter (body)
- **Database**: PostgreSQL (local dev) / Supabase (production)
- **Hosting**: Vercel

## 🎨 Design Philosophy

- Dark (`#0d0d0d`) and Cream (`#F5F0E8`) themes
- PlayStation button colours as accent gradients
- Cinematic hover overlays, auto‑cycling slideshows, and ambient particle effects
- Dead Silence mode — hide everything but the post body
- 10 unique design features from the original notebook
